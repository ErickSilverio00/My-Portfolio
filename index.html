HANGMAN GAME
I did a hangman game that contains 1470 words, in Portuguese, to be generated. This game contains a lot of rules that make it more funny to play. The game can be played on command prompt of any computer that have this program of the game installed.

ROCK PAPER SCISSORS
I did a project, in Portuguese, that was one proof of my college: "Do a game in client/server architecture", so I chose the "rock-paper-scissors" game. On this game, the player plays agains the server, where the player choose "rock", "paper" or "scissor" and the server randomly choose it too. The game consists on 5 rounds and wins the game the one that wins more rounds inside those 5.

CALCULATOR
